Create a descriptive list of at least three terms:

What is Java?
Java is a programming language and a platform. Java is a high level, robust, object-oriented and secure programming language.
What is HTML?
Html Stands for HyperText Markup Language. HTML is a computer language used to create, create web pages and web applications and publish on WWW (World Wide Web).
What is CSS?
This is how CSS is used to style HTML elements. CSS stands for Cascading Style Sheets and is a styling language that allows you to style your web pages in an efficient and easy manner.
What is JavaScript?
JavaScript is a scripting programming language that is used to make desktop widgets, applications, dynamic websites, PDF documents, developing game, etc. All web browsers support JavaScript.
